1. Login ke akun telegram di (web.telegram.org) (Versi lama)

2. Klik pada grup yang ingin di scrape

3. Tentukan jumlah username yang ingin di scrape

4. Klik MULAI SCRAPING

5. Setelah selesai, username akan muncul dibawahnya

How to add bulk members in group?

1. Login your Telegram account on (web.telegram.org) (Old Version)

2. Click on the group that you want to add members

3. Input a list of Telegram username

4. Set the interval between Add (20 to 25 seconds is recommened)

5. Set the numbers of invites

6. Click START ADDING
